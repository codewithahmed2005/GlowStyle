# GlowStyle

A minimal, polished e-commerce prototype built with Flask. Features product browsing, detail views, and WhatsApp integration for orders.

## Features

- ✨ Modern, responsive design optimized for beauty/fashion brands
- 📱 WhatsApp integration with pre-filled messages
- 🖼️ High-quality product imagery with lazy loading
- ⚡ RESTful API endpoints for product data
- 🎨 Elegant UI with smooth animations
